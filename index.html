This app serves primarily to help track where naloxone (Narcan®) is being distributed in Chautauqua County. If you haven’t already done so, please let me know that you’re using this app so I can add your name to the drop down list. (Email link is below.)


To enter an instance of naloxone distribution, tap “Next” (below). This will open a map. You can make the map full screen by tapping ⛶. Zoom into the approximate location where you gave out the naloxone. Drag and drop the red pin onto the approximate location. The pin doesn’t have to be super-precise, just close to the actual location.

You can also search for an address by using the address field below the map.

Tap “Next” again. You’ll then be prompted to answer 6 questions:

  • Source of naloxone: Who provided the naloxone you're distributing? If the agency isn't in the dropdown list, please indicate the source in the "Notes" box, below.


    • Given by: Select your name from the dropdown box.


    • Date given


    • Number of naloxone kits given: This is the number of boxes – each containing 2 doses – that you gave out.


    • Was training provided?: Some people already know how to use it and don’t need/want training. (You can use the videos on the first page (in English and Spanish) as a help for training.)


    • Approximate age: You don’t need to ask the person. Just make your best guess.

    You can add an optional “Note” to provide further information if you’d like.

    If it is not possible to make a separate entry for each person who received naloxone (for example, at a busy event), you can make one entry for multiple kits given. If you have any sense of the demographics of the recipients, add a note to that entry such as: “About half of the recipients were between 20-29, the rest over 50.”

    When finished with the entry, tap “Submit.” This will return you to the app's homepage.
